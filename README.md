# Task 16: Store Sales Forecasting

## Dataset
- `stores_sales_forecasting.csv`

## Steps
1. Loaded dataset and converted date column.
2. Aggregated sales by month.
3. Plotted sales trend and checked seasonality.
4. Split data into train/test.
5. Applied Exponential Smoothing model.
6. Forecasted next 12 months.
7. Computed MAE and MAPE.
8. Exported results to CSV and report.

## Deliverables
- `task16_forecasting.ipynb`
- `forecast_output.csv`
- `forecast_report.txt`
